 Project Overview

TrendPulse is a data pipeline project that collects trending stories from the HackerNews API, categorizes them into meaningful groups, and stores the data in JSON format. This is Task 1 of a multi-step pipeline involving data collection, cleaning, and visualization.

Exposing an API key on GitHub can lead to unauthorized usage, data breaches, and financial loss if the key is misused. Attackers can exploit the key to make requests or access sensitive data.

Logging city names may violate user privacy policies because location data can be sensitive and personally identifiable. Companies restrict such logging to protect user confidentiality and comply with data protection regulations.
